# react-comic-hub

一个基于 `react` 开发的移动端漫画网站

技术栈：react, react-router-dom, react-query, axios, styled-component, vite

代码规范：eslint, prettier, commitlint, commitizen

src 目录：

```
src
├─ assets     - 静态文件
├─ component  - 组件
├─ page       - 页面
├─ hooks      - 自定义hook
├─ util       - 工具
├─ App.jsx    - 根组件
└─ main.jsx   - 入口文件
```
